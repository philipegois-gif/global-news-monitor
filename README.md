##  Exemplo de Saída (Real-time Analysis)
O sistema identifica convergências entre mídias brasileiras (G1) e internacionais (BBC).
Exemplo de processamento:
1. MILITAR (mencionado 3x)
2. PRESO (mencionado 3x)
3. TRUMP (mencionado 3x)

##  Como rodar
1. Clone o repositório
2. Crie o venv: python3 -m venv venv
3. Instale: pip install -r requirements.txt
4. Execute: python3 main.py
